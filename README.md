# Interactive CV - GIS Developer & Digital Twin Specialist

An interactive portfolio website showcasing expertise in GIS development and 3D city digital twins.

## 🌟 Features

- **3D Interactive Background** - Animated city visualization using Three.js
- **Responsive Design** - Works seamlessly on all devices
- **Smooth Animations** - Engaging scroll effects and transitions
- **Project Showcase** - Highlights key GIS and 3D visualization projects
- **Skills Visualization** - Animated progress bars for technical skills
- **Professional Layout** - Modern design with gradient accents

## 🛠️ Technologies

- **Frontend**: HTML5, CSS3, JavaScript
- **3D Graphics**: Three.js
- **Fonts**: Google Fonts (Inter)
- **Hosting**: GitHub Pages

## 💻 Local Development

To run locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/mulfvik/cv-gis.git
   cd cv-gis
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```

3. Navigate to `http://localhost:8000`

## 📝 Customization

To customize the content:

- **Personal Info**: Edit the HTML in `index.html`
- **Styling**: Modify colors and layout in `css/style.css`
- **3D Background**: Adjust animation in `js/three-background.js`
- **Interactions**: Update functionality in `js/main.js`

## 🌐 Deployment

This site is deployed using GitHub Pages:

1. Push changes to the `main` branch
2. Go to repository Settings → Pages
3. Select `main` branch and `/` (root) as source
4. Save and wait for deployment
